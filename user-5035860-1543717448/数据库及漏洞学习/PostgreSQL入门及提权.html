PostgreSQL 是一个自由的对象-关系数据库服务器(数据库管理系统),虚拟机kali

安装

在root权限下修改数据库密码：

  • service postgresql start //启动服务
  • su postgres #切换到数据库用户下
  • psql postgres #进入数据库
  • alter user postgres with password ‘postgres’ #修改数据库的密码为：postgres
