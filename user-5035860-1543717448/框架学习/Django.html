url() 函数
Django url() 可以接收四个参数，两个必选参数：regex、view

  • regex: 正则表达式，与之匹配的 URL 会执行对应的第二个参数 view。
  • view: 用于执行与正则表达式匹配的 URL 请求。
  • kwargs: 视图使用的字典类型的参数。
  • name: 用来反向获取 URL。
